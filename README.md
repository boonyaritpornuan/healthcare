# โครงการ Healthcare

## ภาพรวม
โครงการนี้พัฒนาด้วย Spring Boot เพื่อรองรับสังคมผู้สูงอายุและการทำงานเชิงรุกของอาสาสมัครสาธารณสุขและแพทย์ในชุมชนของประเทศไทย โดยมีการใช้งานเทคโนโลยี HTML และ Java เป็นหลัก

## คุณสมบัติ
- การจัดการสุขภาพเชิงรุก
- รองรับการดูแลผู้สูงอายุ
- การประสานงานของอาสาสมัครสาธารณสุขในชุมชน

## เทคโนโลยีที่ใช้
- **Java**: 62.1%
- **HTML**: 37.9%

## การเริ่มต้นใช้งาน
ในการเริ่มต้นใช้งานโครงการนี้ ให้คลังโค้ดและปฏิบัติตามขั้นตอนด้านล่าง:

```bash
git clone https://github.com/boonyaritpornuan/healthcare.git
cd healthcare
```

## การรันแอปพลิเคชัน
1. ตรวจสอบว่ามีการติดตั้ง Java บนเครื่องของคุณ
2. ไปยังไดเรกทอรีของโครงการ
3. รันแอปพลิเคชันโดยใช้คำสั่งด้านล่าง:

```bash
./mvnw spring-boot:run
```
